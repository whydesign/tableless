1. Setup

# Setup

  1. Downloade the latest release from the Github repository: tableless@github
  2. Unzip and paste all files to your project root
  3. Add tableless.css to your html head
  4. Add the basic HTML mockup to your project like this
  5. Customize the global variables unter src/tableless.css

2. Settings

# Table Classes

Class definition for table view. Initialise the table view with .tbl-less and define the viewport by the .tbl-table- prompt.

Class
Description
.tbl-less
Initialise the tableless css
.tbl-table-s
Start table view on small viewport. default 0px
.tbl-table-m
Start table view on medium viewport. default 640px
.tbl-table-l
Start table view on large viewport. default 1024px
.tbl-table-xl
Start table view on x-large viewport. default 1200px
.tbl-table-xxl
Start table view on xx-large viewport. default 1440px
.tbl-divider
Add a border divider to each column in table view.
.tbl-spacer
Add a row spacer in table view.

# Card Classes

Class
Description

.tbl-card-2@s
Display 2 cards on small viewport.
.tbl-card-3@s

.tbl-card-4@s
Display 4 cards on small viewport.
.tbl-card-5@s
Display 5 cards on small viewport.

Display 6 cards on small viewport.

You can combine the card classes width the viewport definition by adding the @X prompt.

Description

Display 2 cards in row on x-large viewport.
.tbl-card-2@xxl
Display 2 cards in row on xx-large viewport.

You can combine the text alignment classes width the viewport definition by adding the @X prompt.

# Global Variables

Define your custom global variables. You can define them in src/tableless.scss

Variable
Description
Default
$tbl-head-font-weight
Define the table header font weight.
800
$tbl-font-color
Define the font color.
#333333
$tbl-font-size
Define the font size.
0.75em
$tbl-border-style
Define the border style.
1px solid
$tbl-border-color
Define the border color.
#B2B2B2
$tbl-img-width
Define the max image width.
100px
$tbl-card-column-gap
Define the gap between columns.
25px
$tbl-card-row-gap
Define the gap between rows.
25px
$tbl-card-radius
Define the border radius for card view.
5px
$tbl-card-shadow
Define the shadows for card view.
0 2px 8px rgba(0,0,0,.08)
$tbl-card-border
Define the border for card view.
$tbl-border-style lighten($tbl-border-color, 20%)
$tbl-card-padding
Define the padding between the card views.
0 0 20px 0
$tbl-button-font-size
Define the button font size.
0.74em
$tbl-button-padding
Define the the button padding.
0 18px
$tbl-button-line-height
Define the button height.
28px
